# 🎮 NEXUS BOOST

> Fortnite Performance Control Center for Windows + Linux + GeForce Now

---

## 🚀 Overview

NEXUS BOOST is a desktop performance optimization and game management tool designed to improve system responsiveness and stability for Fortnite.

It focuses on **safe system-level improvements**, not game modification, including:
- Background process optimization
- CPU/GPU performance tuning
- Game launch profiles
- Real-time hardware monitoring
- Low-latency mode for competitive play

Works on:
- 🪟 Windows PCs
- 🐧 Linux systems
- 🌐 GeForce Now (browser optimization mode)

---

## ⚡ Features

### 🎯 Boost Mode
One-click system optimization:
- Enables high performance power mode
- Boosts Fortnite process priority
- Disables unnecessary background tasks
- Applies competitive gaming configuration

---

### 📊 Live Performance Dashboard
Real-time stats:
- FPS (estimated/overlay-supported)
- CPU usage
- GPU usage
- RAM usage
- System load tracking

---

### 🎮 Fortnite Launcher Profiles
Pre-configured performance modes:
- Low-End Mode (stability first)
- Competitive Mode (high FPS priority)
- Balanced Mode (visual + performance mix)

---

### 🌐 GeForce Now Support Mode
Optimizations for cloud gaming users:
- Browser performance tuning
- Tab/resource cleanup suggestions
- Network latency testing tools
- Lightweight overlay HUD

---

### 🧠 Smart Optimization Engine
Automatically detects:
- Low RAM systems
- Weak GPU performance
- Laptop battery mode
- Thermal throttling risk

Then applies recommended settings.
